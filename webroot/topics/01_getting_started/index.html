Getting started

Welcome

  • Foundation is a responsive frontend framework
  • It is CSS (and SASS), JavaScript and HTML
  • Developed with an eye on UX best practice and web standards accessibility
  • Has offical templates, building blocks and foundation kits (bundles of building blocks) which you may use to speed up your development
  • It built with SASS which is "CSS with superpowers" and compiles to plain CSS but can be written using variables, mixins and nesting to save typing and arrange better
  • Used to be a framework for sites, apps and emails. But this is now streamlined to sites and emails frameworks (with some apps framework stuff going into sites and apps framework being generally available as a playgound experiment)

Installation

  • Can install with a package manager (npm, bower, composer etc) which is good if you need or want the SASS source
  • Can git clone the official github repo (but this method not mentioned on 6.4 docs?)
  • Download (full, lightweight or custom options) - good if you don't need or want the SASS
  • CDN
  • "Foundation CLI" which is an npm meta installer / meta package

Starter projects

  • Foundation CLI can install some different project templates
  • Basic template and ZURB template
  • Both templates use "Gulp" build system (to automate Sass compilation, JavaScript processing and etc)
